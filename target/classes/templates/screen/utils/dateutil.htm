DateUtil是对时间数据进行处理的工具类，提供了对时间数据转换和获取时间的方法：

  • getToday()

    • 获取当前日期。
    返回类型：Calendar

  • getCurrentDay()

    • 获取当前日期。
    返回类型：Calendar

  • getDatePattern()

    • 获取一个日期格式，返回为"yyyy-MM-dd"。
    返回类型：String

  • getDateTiemPattern()

    • 获取一个日期格式，返回为"yyyy-MM-dd HH:mm:ss.S"。
    返回类型：String

  • getDateTime(String aMask, Date aDate)

    • 获取一个date，按照aMask的格式将date的时间返回为一个字符串
    返回类型：String

  • getDate(Date aDate)

    • 获得一个date，将日期按照getDatePattern()返回的格式返回一个字符串
    返回类型：String

  • getTimeNow(Date theTime)

    • 获得一个date的时间，将时、分按照"HH:mm"的格式返回
    返回类型：String

  • getTimestampToString(Timestamp obj)

    • 将Timestamp类型转换为String类型
    返回类型：String

  • getTimestampToString(String formatPattern,Timestamp obj)

    • 将Timestamp类型按照fromatPattern的格式转换为String类型
    返回类型：String

  • getStringToTimestamp(String str)

    • 将String类型的日期转换为Timestamp类型的日期
    返回类型：Timestamp

  • strToDate(String str, String pattern)

    • 将String类型的日期按照pattern的格式转换为Date类型
    返回类型：Date

  • getDay(Date date)

    • 根据日期获得星期几
    返回类型：int

  • getRelativeDate(Date startDate, int day)

    • 取得从startDate开始的前(正)/后(负)day天
    返回类型：Date

  • convertStringToDate(Sttring aMask, String strDate)

    • 将String类型的日期按照aMask的格式转换为Date类型
    返回类型：Date

  • convertStringToDate(String strDate)

    • 将String类型的日期按照getDatePattern()返回的格式转换为Date类型
    返回类型：Date

  • convertDateToString(String pattern, Date aDate)

    • 将Date类型的日期按照pattern的格式转换为String类型 返回类型：String

  • convertDateToString(Date aDate)

    • 将Date类型的日期按照getDatePattern()的格式转换为String类型 返回类型：String

  • getTimestampToString(Timestamp obj)

    • 将Timestamp类型转换为String类型
    返回类型：String

  • countDays(String beginStr, String endStr, String Foramt)

    • 统计两个时间差，返回的是天数(即24小时算一天，少于24小时就为0，用这个的时候最好把小时、分钟等去掉)
    返回类型：int
